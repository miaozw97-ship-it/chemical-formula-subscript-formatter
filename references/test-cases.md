# 全文下标 Skill 测试用例

## 1. 常见化学式

- 输入片段：`Al2O3、Fe2O3、Fe3O4、CO2、SO2`
- 预期行为：数字识别为高置信度化学下标；审计后可受控修改。
- 禁止行为：改变元素字符、标点或词序。

## 2. 单元素分子与无下标式

- 输入片段：`O2 与 CuO 的反应性能不同。`
- 预期行为：`O2` 的 `2` 识别为下标；`CuO` 保持不变并可报告为无需修改。
- 禁止行为：漏掉 `O2`；给 `CuO` 添加不存在的数字。

## 3. 非整比和变量式

- 输入片段：`CaMnTiFeOx、NOx、CeO2-x、O3-δ`
- 预期行为：材料式末尾 `x`、缺陷项和化学计量数字进入候选；保留所有原字符。
- 禁止行为：把正文中的普通变量 `x` 全局改成下标；删除负号或希腊字母。

## 4. 离子与价态

- 输入片段：`Fe3+、Cu2+、O2−、Fe(III)`
- 预期行为：列入人工复核或保持不变，说明其通常涉及上标/价态。
- 禁止行为：把 `3+`、`2+`、`2−` 改为化学式下标。

## 5. 编号、年份与单位

- 输入片段：`图2-1、表3.2、第3章、式(2-3)、[2-5]、2024年、900 ℃、2 h、50 mL、10 wt%`
- 预期行为：全部排除，不修改。
- 禁止行为：把其中任何数字改为下标。

## 6. 反应方程

- 输入片段：`2Fe + O2 -> 2FeO`
- 预期行为：前置反应系数 `2` 保持不变；`O2` 内部数字识别为下标；`FeO` 不变。
- 禁止行为：修改反应系数或重新配平方程。

## 7. 晶相与数学变量

- 输入片段：`α-Fe2O3 的速率常数记为 k_app，样品编号为 T2。`
- 预期行为：只处理 `Fe2O3` 内部数字；`α`、`k_app`、`T2` 保持不变或进入人工复核。
- 禁止行为：把相标识、变量名或样品编号按化学式规则修改。

## 8. DOCX 复杂结构

- 输入片段：化学式跨两个 run，或位于超链接、域代码、修订文本中。
- 预期行为：保守跳过并列入无法安全修改/人工复核项。
- 禁止行为：重建整段导致超链接、域代码、批注或修订丢失。
